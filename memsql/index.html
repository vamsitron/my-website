MySQL

Official Website: https://www.memsql.com/
License: Closed Source
Trial: https://www.memsql.com/download/

MemSQL is a distributed, in-memory, SQL database management system. It is a relational database management system (RDBMS). It compiles Structured Query Language (SQL) into machine code, via termed code generation. On April 23, 2013, MemSQL launched its first generally available version of the database to the public

Main features

  • MemSQL can store database tables either as rowstores or columnstores.
  • MemSQL database is a distributed database implemented with aggregators and leaf nodes.
  • MemSQL ships with an installation, management and monitoring tool called MemSQL Ops.
  • MemSQL gives users the ability to install Apache Spark as part of the MemSQL cluster
  • MemSQL can also store columns in JSON format, and supports geospatial datatypes and operations.
  • MemSQL cluster can be configured in "High Availability" mode, where every data partition is automatically created with master and slave versions on two separate leaf nodes.
